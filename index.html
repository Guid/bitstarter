G-Code Anayalzer is designed to help CNC programmers and operators get an insightful understanding of their program. It can also run a CNC machine via serial.

Verify
  • View program flow
  • Mouse over elements to see partial toolpath
  • Jump to code location from toolpath
  • View modal states
  • Measure distances
  • Generate setup sheets (MS Word, OpenOffice)
  • Datum display
  • Support anaglyph 3D

DNC
  • Drip feed/send programs via RS232 or Bluetooth
  • Realtime toolpath animation
  • Resume from any position in the program (modals are reconstructed and loaded)
  • Supports subprograms (M98) and block skip (/)
Edit
  • Adjust feedrate by contouring, plunging and ramping (on-the-fly adjustment while drip feeding).
  • Shift Z coordinates
Full editor, and more hardware support is planned.
